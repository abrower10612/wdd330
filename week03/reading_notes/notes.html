Chapter 5: Objects

Object Literals

Objects are basically dictionaries that store related information. If a property's value is a function, it is known as a method.

Creating Objects

To create an object literal, simply enter a pair of curly braces.

Dot notation is the best way to access the properties of an object. If you try to access a property that doesn't exist, undefined will be returned.

A property can be a concatenation of strings inside of square brackets, which is an example of a computed property.A new property can be added t oan object using a symbol as a key if the square bracket notation is used.

Calling Methods

To call an object's method we can also use dot or bracket notation. Calling a method is the same as invoking a function, so parentheses need to be placed after the method name.

Checking if Properties or Methods Exist

All objects have a method called hasOwnProperty(). The 'in' operator can be used to check whether an object has a particular property.

Finding all the Properties of an Object

Loop through all object properties and methods with for in loop.You can also use Object.values() that will return an array of all thet object's values. Object.entries() returns an array of key-value pairs.

New properties are added to an object by simply assigning a value to the new property. Be aware though that an objet is not an ordered list like an array, set, or map, so you should never rely on the properties to be in a certain order.

The value of an object's properties can also be changed using assignment.

Removing Properties

Any property can be removed from an object using the delete operator (e.g. delete superman.fly).

Nested Objects

Objects can contain other objects called nested objects. The values in nested objects can be accessed with dot notation by using each property name in order. Be aware that objects are copied by reference, which means that the referenced object will be changed if the cloned object is changed. They both point to the same space in memory.

Objects can be passed to functions as parameters. In this case arguments can be provided by name and in any order. Values can be overridden by simply naming the parameter and setting a different value to it.

The keyword 'this' refers to the object that it is within. It can be used inside methods to gain access to the object's properties.

Namespacing

Naming collisions occur when the same variable or function name is used for different purposes by code sharing the same scope. The object literal pattern can be used to create a namespace for groups of related functions. This is done by creating an object literal that serves as the namespace, then adding any values as properties of that object, and any functions as methods. So basically this is the practice of creating a new scope that will help avoid naming collisions.

Built-in Objects

Aside from arrays and functions, JavaScript has a number of other built-in global objects that can be accessed from anywhere in a program.

JSON stands for JavaScript Object Notation. It is often used to exchange information between web services. It hits the sweet spot between being human and machine readable. In JSON, property names must be double-quoted. Also, permitted values are double-quoted strings, numbers, true, false, null, arrays and objects. And functions are not permitted values. The parse() method takes a string of data in JSON format and returns a JavaScript object. The stringify() method does the opposite, taking a JavaScript object and returning a string of JSON data. A space argument can be added that will add new lines between each key-value pair, which is useful when displaying the results in a browser.

The Math object has eight properties that represent a mix of commonly used math constants: Math.PI, Math.SORT2, Math.SQRT1_2, Math.E, Math.LN2, Math.LN10, Math.LOG2E, and Math.LOG10E.

The Math object also has several methods to carry out a variety of useful mathematical operations. The Math.abs() method returns the absolute value of a number. The Math.ceil() method will round a number up to the next integer, or remain the same if it is already an integer. The Math.floor() will round a number down to the next integer or remain the same if it is an integer. The Math.round() method will round a number to the nearest integer. The Math.trunc() method returns the integer-part of a number.

The Math.exp() method will raise a number to the power of Euler's constant. The Math.pow() method will raise any number to the power of another number. The Math.sqrt() method returns the positive square root of a number. The Math.cbrt() method returns the cube root of numbers. The Math.hypot() method returns the square root of the sum of the squares of all its arguments.

The Math.log() method returns the natural logarithm of a number.

The Math.max() method returns the maximum number from its arguments. (e.g. 3 is the max out of 1, 2, and 3).

Trigonometric Functions:

The Math.sin() returns the sine of an angle. The Math.cos() returns the cosine of an angle. The Math.tan() returns the tangent of an angle. The Math.asin() returns the arcsine of a number. The Math.acos() returns the arccosine of a number. The Math.atan() returns the arctangent of a number. Methods for the hyperbolic functions, sinh(), cosh(), and tanh() were also added as all we their inverses.

Random Numbers

The Math.random() method is used to create random numbers (e.g. '6 * Math.random()'' will generate a random number between 0 and 6). In order to generate a random integer: 'Math.floor(6 * Math.random())''.

The Date Object:

A constructor function is used to create a new date object using the 'new' operator. Remember that computer programs start counting at zero, so January is 0, Febuary is 1, and so on up to December, which is 11. An alternative is to use a timestamp, which is a single integer argument that represents the number of milliseconds since the Epoch (1st january 1970).

Getter Methods:

Once you've created a date object it will have access to all the getter methods. Most methods have two versions, one that returns information for local time and one that returns information for Coordinated Universal Time (UTC).

Setter Methods:

Setter methods can be used to change the value of the date held in a Date object.

The RegExp Object:

A regular expression (RegExp) is a pattern that can used to search strings for matches to the pattern. 'Find and replace' is a common type of operation (e.g. /[a-zA-Z]+ing$/).

Creating Regular Expression:

Aside from the example above, you cacn create a new instance of the RegExp object using the new operator and a constructor function (e.g. const pattern = new RegExp('[a-zA-Z]+ing');).

RegExp Methods:

The test() method can be used to see if a string matches the regular expression pattern. It returns true if the pattern is in the string, and false if it isn't.

Character Groups

Groups of characters can be placed together inside square brackets. A dash includes a sequence of characters. '/^' negates the sequence if placed before it.

Regular Expression Properties:

Regular expressions are objects and have the following properties: global (Makes pattern return all matches. The pattern only looks for the first occurrence of a match), ignoreCase (makes the pattern case-insensitive), and multiline (makes the pattern multiline). The following flags can be placed after a regular expression literal to change the default properties: g (global property true), i (ignoreCase property true), and m (multiline property true).

Special Characters:

Characters that have a special meaning, AKA metacharacters.

Modifiers:

Can be placed after a token to deal with multiple occurrences of that token. Any special characters or modifiers can be escaped using a backslash.

Greedy and Lazy Modifiers:

All modifiers mentioned above are greedy, which means they will match the longest possible string. They can be made into lazy modifiers that match the shortest possible string by adding an extra '?' after the modifier.

String Methods:

The split() method can accept a regular expression that's used to split a string into the separate elements of an array.

Matched Groups:

Capturing groups are sub-patterns that are created inside a regular expression by placing them inside parentheses.
